Запуск проекта осуществляется на компьютере из интегрированной среды
разработки visual Studio
Необходима версия минимум 2019 года
Также встроенная СУБД SQLiteStudio

1. Скачиваем проект по ссылке: https://github.com/IlyaStratievskij/MyProject/tree/InventoryManagmentSystem
2. Открываем папку InventoryManagmentSystem 
3. 2 клика ПКМ по файлу Inventory_Managment_System.sln
4. Открываем обозреватель решений
5. В окошке "Добавить подключение", в поле "Имя файла БД" 
выбираем файл в папке Inventorydb.mdf и нажимаем "ОК"
6. Нажимаем "Запуск программы" в Visual Studio.
